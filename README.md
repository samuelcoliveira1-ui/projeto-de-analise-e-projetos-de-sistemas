## Objetivo do Projeto

O **MP3 Search** busca integrar um sistema de busca do YouTube ou outras plataformas para baixar localmente músicas e vídeos de forma totalmente gratuita. 

A ideia principal é evoluir o sistema para termos um mecanismo de busca integrado diretamente no próprio aplicativo. Com isso, os usuários poderão pesquisar seus conteúdos e mídias diretamente no app para realizar o download, **eliminando completamente a necessidade de copiar e colar links externos**.

**Principais objetivos:**
* Centralizar os trabalhos e organização do código;
* Versionar e controlar as alterações do script de download;
* Desenvolver uma interface intuitiva que dispense o uso de URLs de terceiros;
* Desenvolver habilidades técnicas em Python, UI/UX e metodologias ágeis.

## Tecnologias Utilizadas

* **Python:** Linguagem utilizada para construir o motor de busca e automação do download das mídias.
* **VS Code:** Ambiente de desenvolvimento (IDE) utilizado para a escrita e execução do código via console.
* **Figma:** Ferramenta utilizada para desenhar o template e planejar a interface visual do aplicativo.
* **Git & GitHub:** Utilizados para o controle de versão, histórico de alterações e compartilhamento do projeto.
